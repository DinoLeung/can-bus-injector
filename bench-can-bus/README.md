# Bench top can bus

This is to save me from getting actual can signal from the car while writing code. 

It's purely vibe coded, making [FakeSubaruBRZ.ino](https://github.com/timurrrr/arduino-CAN/blob/master/examples/FakeSubaruBRZ/FakeSubaruBRZ.ino) compatible with ESP32-CAN-X2, producing CAN data on CAN 1.
